About Contact Manager

Introduction

Contact Manager is a user-friendly web application designed to simplify contact management. Built with modern technology and a focus on user security, it offers a seamless experience for organizing, updating, and accessing personal and professional contacts in one secure location.

Core Features

  • Secure User Authentication: Protects user data with Spring Security.
  • Full CRUD Functionality: Allows for easy addition, update, and deletion of contacts.
  • User Profiles: Customizable user profiles with profile pictures and personal information.
  • Role-Based Access Control: Provides access restrictions based on user roles.

Technologies Used

  • Backend: Java with Spring Boot (including Spring Security, JPA)
  • Frontend: Thymeleaf for dynamic rendering
  • Database: MySQL for reliable data storage
  • Build Tool: Maven for dependency management

Motivation and Goals

The idea for Contact Manager stemmed from the need for a secure and accessible way to manage contacts. The goal was to build a solution that balances data privacy, usability, and performance, providing users with a tool they can trust for organizing their personal information.

Future Enhancements

  • Enhanced Search and Filter Options
  • Integration with Third-Party APIs for email and contact sync
  • Customizable Contact Groups
  • Mobile-Friendly Design
